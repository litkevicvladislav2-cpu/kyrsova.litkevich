# Keyboard Event Tracker (C++)
Програма для відстеження натискань клавіш у Windows.

## Функціонал
- Відстеження клавіш
- Запис у файл log.txt

## Технології
- C++
- WinAPI

## Запуск
Скомпілювати у Visual Studio та запустити від імені адміністратора.

## Примітка
Проєкт створений з навчальною метою.
